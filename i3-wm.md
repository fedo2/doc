# ⌨️ Nejčastější defaultní zkratky v i3‑wm

| Akce | Zkratka | Popis |
|------|---------|--------|
| **Otevřít terminál** | `$mod+Enter` | Spustí výchozí terminál (obvykle `i3-sensible-terminal`). |
| **Zavřít okno** | `$mod+Shift+q` | Pošle „kill“ aktuálnímu oknu. |
| **Přepínání fokusů (šipky)** | `$mod+h/j/k/l` | Pohyb mezi okny vlevo/dolů/nahoru/vpravo (vim styl). |
| **Přepínání fokusů (šipky 2)** | `$mod+Left/Down/Up/Right` | Alternativní způsob pohybu mezi okny. |
| **Změna velikosti okna** | `$mod+r` | Přepne do resize módu (pak šipky nebo h/j/k/l). |
| **Dlaždicový / plovoucí mód** | `$mod+Shift+Space` | Přepíná okno mezi tiled a floating. |
| **Focus mezi floating/tiled** | `$mod+Space` | Přepíná fokus mezi plovoucími a dlaždicovými okny. |
| **Fullscreen** | `$mod+f` | Přepne okno do fullscreen režimu. |
| **Horizontální split** | `$mod+h` | Vytvoří horizontální rozdělení (nové okno vedle). |
| **Vertikální split** | `$mod+v` | Vytvoří vertikální rozdělení (nové okno pod). |
| **Přesun okna** | `$mod+Shift+h/j/k/l` | Přesune okno vlevo/dolů/nahoru/vpravo. |
| **Přesun okna na workspace** | `$mod+Shift+1..0` | Pošle okno na workspace 1–10. |
| **Přepnutí workspace** | `$mod+1..0` | Přepne na workspace 1–10. |
| **Restart i3** | `$mod+Shift+r` | Reloadne konfiguraci a restartuje WM bez ukončení aplikací. |
| **Ukončit i3** | `$mod+Shift+e` | Otevře dialog pro ukončení i3. |
